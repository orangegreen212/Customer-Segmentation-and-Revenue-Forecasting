# Driving Growth for Online Retail: Customer Segmentation and Revenue Forecasting

## Project Description
This project analyzes transactional data from a UK-based online retailer. It creates customer segments (Champions, At-Risk, etc.) using RFM analysis and K-Means clustering, and forecasts future sales using the Prophet model.

## Folder Structure
- /data - Contains the dataset (online_retail.csv)
- /notebooks - Jupyter Notebook with the code
- /dashboard - Tableau Workbook (.twbx)
- /presentation - PowerPoint slides

## Technologies Used
- Python (Pandas, NumPy, Scikit-learn, Prophet)
- Tableau (for visualization)

## How to Run
1. Install requirements: pip install pandas numpy matplotlib seaborn scikit-learn prophet
2. Open "Customer_Segmentation.ipynb" in Jupyter Notebook.
3. Run all cells to see the analysis and forecasting results.
4. Open the Tableau file to interact with the dashboard.
